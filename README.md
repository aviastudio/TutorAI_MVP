# Tutor AI MVP

Tutor AI helps students truly understand concepts through deep, personalized learning adapted to each user’s level and style.

It explains concepts step-by-step, answers contextual questions, and creates personalized quizzes.

We’re currently testing it in schools, with the goal of helping anyone learn independently in the future.
